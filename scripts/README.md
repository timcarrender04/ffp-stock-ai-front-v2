# FFP Stock AI Frontend - Scripts

This directory contains utility scripts for the project.

## Available Scripts

### Development Scripts

- **[start-dev.sh](./start-dev.sh)** - Start the development server

### Diagnostic & Maintenance Scripts

- **[DIAGNOSTIC.sh](./DIAGNOSTIC.sh)** - Run diagnostic checks on the application and environment

### Quick Start Scripts

- **[QUICK_START.sh](./QUICK_START.sh)** - Quick start script to set up and run the application

### Git & Deployment Scripts

- **[PUSH_TO_GIT.sh](./PUSH_TO_GIT.sh)** - Push changes to Git repository with proper checks

## Usage

Make sure scripts are executable:

```bash
chmod +x scripts/*.sh
```

Run any script from the project root:

```bash
./scripts/script-name.sh
```

Or from the scripts directory:

```bash
cd scripts
./script-name.sh
```

---

**Note:** Some scripts may require environment variables to be set. Check the script contents or the main project documentation for requirements.
